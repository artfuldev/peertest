# Peer Test
Simple one-to-one audio/video call app using PeerJs.

You can find a working example here:
https://simple-av.herokuapp.com

#How to set up:

1. Clone Repo
2. npm install
3. cd public
4. bower install
5. change mongodb url in /db/schema/index.js to point to your local mongodb server
6. node app.js
